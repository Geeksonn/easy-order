## Easy Order 
Easy Order is a web-application dedicated to easy the order of goods. 
Within the configuration, you define the stock that you have, including the price (and currency) of each item + a small image representing the item. 
In the main page, you will have all your items displayed and you'll be able to tap/click on items to add them in the cart. 
Cart shows the total number of items and total price. 
Then, either you cancel the whole order or you validate it. 

## Technical Stack
Easy Order is based on NextJS framework. The whole application (front-end + backend) is developped within this project. The back-end publishes API that uses an external MongoDB (hosted on Atlas).
